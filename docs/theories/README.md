### [RAG and Semantic Search](./rag-semantic-search.md)
**Comprehensive guide to Retrieval-Augmented Generation and Semantic Search**

Topics covered:
- What is RAG and why it matters for agricultural advisory systems
- Semantic search vs keyword search with agricultural examples

- Complete RAG architecture explanation
- Implementation details from this thesis
- Benefits, limitations, and future directions

**Key sections**:
- Real-world examples using crop queries
- Performance comparisons (semantic vs keyword)
- Step-by-step data flow through the system
- Relevance to agricultural information access problems

**Target audience**: Thesis reviewers, developers, researchers interested in domain-specific RAG applications
